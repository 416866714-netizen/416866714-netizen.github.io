# 老谭话术自动化｜项目原文档

项目编号：老谭第四个项目
项目名称：老谭话术自动化
线上入口：`https://laotanuwen.com/tools/scripts/`
项目目标：把代运营公司积累的大量小红书、微信、客户沟通话术，做成可视化分支树 + AI 智能回复工作台。

## 1. 项目定位

“老谭话术自动化”服务于客户沟通转化，不是内容生产工具。它解决的问题是：客户问了一句话，用户把原话粘进去，系统根据品牌知识库、标准话术库、聊天记录风格库和当前客户阶段，输出最合适的回复。

## 2. v1 功能

1. 客户问题输入：粘贴客户原话。
2. 智能判断客户类型：问价格、质疑效果、要案例、考虑一下、不回复、已成交维护等。
3. 话术分支树：点击不同客户状态，展开下一步回复策略。
4. 三类知识库：品牌知识库、标准话术库、我的聊天风格库。
5. AI 综合回复：推荐回复、温柔版、强势版、微信版、小红书私信版、下一步追问、禁用话术。
6. 风格学习：上传聊天记录截图，GPT OCR 提取用户语气、节奏、常用句式、成交方式。
7. 历史记录：保存每次客户问题和输出结果。
8. 一键复制：复制推荐回复或不同版本。
9. 密码保护：内部工具。

## 3. 与小红书自动化的关系

小红书自动化负责内容生产；话术自动化负责沟通转化。两个项目独立，但共用底层 GPT、图片 OCR、品牌知识库理念和长期数据沉淀方向。
